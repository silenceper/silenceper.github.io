      Petdex: Find an Animated Companion for Codex

      May 6, 2026

      One-line Positioning

      Petdex is a public gallery of animated companions for Codex. It lets you browse, preview, collect, and install pixel-style companions so your local Codex experience feels a little more personal and playful.

      Basic Info

      ItemInfo
      Websitehttps://petdex.crafter.run/
      GitHubhttps://github.com/crafter-station/petdex
      PositioningCodex-compatible animated pets public gallery
      Current scale870+ pets shown on the homepage, with 600+ GitHub stars
      Quick install examplenpx petdex install boba

      What Problem Does It Solve?

      Command-line tools like Codex are efficient, but they can also feel a bit cold: text, tasks, logs, and results fill the window, while the environment itself has very little personality.

      Petdex does something simple and charming. It turns animated Codex companions into a public gallery, so users can browse it like an avatar library or sticker collection and pick a pet for their own Codex setup.

      It solves three small but real problems:

      1. Centralized discovery: instead of hunting for pet packs across scattered links, you can browse them in one place.
      2. Preview before installing: each companion has visuals, states, descriptions, tags, and metadata.
      3. Clear installation path: the site gives you a direct install command, making the first try low-friction.

      This is not a tool for making the model more capable. It is a small layer of personality and ritual around a developer workflow.

      Core Features

      1. Browse the Pet Gallery

      The homepage presents a large set of companion cards. Each card includes details such as number, name, category, description, tags, author, and install data.

      For example, Boba is a tiny otter sipping bubble tea. Duo, Doraemon, Noir Webling, and other companions are presented in pixel-art or chibi-style forms.

      That makes Petdex feel more like a living pet index than a plain file directory.

      2. Install with One Command

      The homepage shows a direct install example:

      npx petdex install boba

      The interaction is straightforward: find a companion you like, copy the command, and install it.

      3. Featured Collections

      Petdex also includes collections, with featured sets on the homepage. Examples include:

      • GRAYCRAFT: an original mech-style character set
      • Meme Lords: meme-inspired characters such as Pepe, Wojak, and Doodlebob
      • Anime Heroes: anime-inspired companion designs

      Collections make browsing more thematic and give creators a way to present related designs together.

      4. Community Submission

      Petdex includes a Submit entry point. Its README also mentions browser-based validation and submission for community pet packages.

      That means Petdex is not just a read-only catalog. It is designed to grow through community contributions.

      5. Open-source Maintenance

      The Petdex GitHub repository is public. The README describes it as a Codex-compatible animated pets gallery and lists several core capabilities:

      • Browse approved pet packs
      • Preview every animation state
      • Download individual ZIP packages
      • Download the full gallery pack
      • Validate and submit community pet packages in the browser

      From the repository structure, pet packages live under public/pets, while downloadable archives are generated under public/packs.

      Who Is It For?

      Petdex is a good fit for:

      • people who use Codex often and want a more personal local environment;
      • developers who like pixel art, chibi characters, and desktop-pet-style details;
      • users who want their toolchain to feel a little less sterile;
      • creators who want to design and share Codex companions.

      If you only care about model quality, task success rate, or engineering automation, Petdex is not essential. But if you spend a lot of time working with Codex in a terminal, it can make that daily experience feel lighter.

      Quick Start

      Open the website:

      https://petdex.crafter.run/

      Pick a companion, such as Boba, and run:

      npx petdex install boba

      You can also keep exploring more pets, collections, and creators on the site.

      To learn more about the project itself, visit the GitHub repository:

      https://github.com/crafter-station/petdex

      Conclusion

      Petdex is not a large or complicated tool. It is more like a playful entry point in the Codex ecosystem: a public index where companions can be browsed, previewed, installed, and submitted.

      Its value is not in making Codex smarter. Its value is in making the experience of using Codex more personal, lighter, and more like a workspace that the community can decorate together.
